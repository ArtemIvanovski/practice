Fenêtre de paramètres

Principaux

Sur cet onglet, vous pouvez définir la taille des miniatures des images, ainsi que le nombre maximal d'images affichées sous forme de miniatures. Évitez de rendre les miniatures des images trop grandes, car cela peut augmenter la consommation de mémoire et ralentir le fonctionnement du programme.

Pour accélérer l'ouverture de la galerie et une navigation plus rapide, le programme propose un mode d'affichage de la galerie sous forme de tableau de noms de fichiers avec une seule miniature de l'image sélectionnée. Ce mode est activé automatiquement lorsque le nombre d'images dans la galerie dépasse le nombre spécifié dans cet onglet de paramètres.

Comparaison

Sur cet onglet, vous pouvez définir les paramètres pour la recherche d'images similaires : méthodes de hachage (aHash, pHash, gHash, dHash), seuil de similitude. Le seuil de similitude est l'un des paramètres les plus importants lors de la recherche. Les résultats de la recherche ne montreront que les images dont le seuil de similitude est supérieur à la valeur définie. Si vous souhaitez trouver des duplicatas exacts d'images, vous devez sélectionner la valeur 100. Réduire ce paramètre augmentera le nombre de paires dans les résultats. Notez que l'activation de toutes les méthodes de hachage augmentera le temps de comparaison.

Formats de fichiers

Onglet pour choisir les formats de fichiers à utiliser lors du fonctionnement du programme. Si des images d'autres formats sont présentes dans le répertoire sélectionné et ne sont pas sélectionnées dans le programme, elles seront simplement ignorées.

Langue

Onglet pour choisir la langue de l'interface du programme. Par défaut, le russe est sélectionné, mais vous pouvez le changer si vous le souhaitez.
