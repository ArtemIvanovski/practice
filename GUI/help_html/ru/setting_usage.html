Окно настроек

Основные

На этой закладке можно установить размеры превьюшек картинок, максимальное количество изображений, отображаемых в виде превьюшек. Старайтесь не делать превьюшки картинок слишком большими, так как при увеличении размера расходуется больше места в памяти и соответственно программа начинает работать медленнее.

Для ускорения открытия галереи и более быстрой навигации по ней, в программе предусмотрен режим отображения галереи в виде таблицы имён файлов с одной превьюшкой выбранного изображения. Этот режим автоматически включается, когда число изображений в галерее превышает указанное число на этой вкладке настроек.

Сравнение

В этой вкладке можно установить параметры для поиска похожих изображений: методы хэширования (aHash, pHash, gHash, dHash), порог схожести. Порог схожести - один из самых значимых параметров при поиске. В результатах поиска будут только те изображения, порог схожести у которых больше установленного значения. Если Вы хотите найти полные дубликаты изображений, то необходимо выбрать значение 100. Уменьшение этого параметра приводит к увеличению большему числу пар в результатах. Необходимо также понимать, что включение всех методов хэширования увеличит время сравнения.

Форматы файлов

Закладка для выбора форматов файлов при работы программы. Если в выбранном катологе изображений будут присутствовать изображения других форматов, не выбранных в программе - они будут просто пропущены.

Language

Закладка для выбора языка интерфейса программы. По умолчанию должен быть выбран русский, при желании его можно изменить.
