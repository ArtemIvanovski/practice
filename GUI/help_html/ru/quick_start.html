Быстрый старт

В программе существует возможность выбора сравниваемых изображений: внутри одной группы изображений (галереи) и между двумя группами изображений. Галерея представляет собой группу изображений, в которой может находиться сколько угодно изображений, даже если они расположены на разных дисках.

Варианты сравнения

Рассмотрим два варианта сравнения:

  • Внутри одной группы

    Программа будет просматривать все изображения данной галереи и искать похожие в этой же группе. Каждое изображение галереи будет сравниваться со всеми остальными изображениями этой

  • Между двумя группами изображений

    Поиск похожих изображений будет происходить не внутри одной группы, а между галереями. Программа будет искать похожие изображения из первой группы, просматривая все изображения второй. В этом случае поиск внутри групп не производится, каждое изображение из первой галереи сравнивается с каждым изображением из второй галереи.

Шаг 1. Добавление элементов в галереи

На этом шаге необходимо выбрать папку с изображениями. Для этого нажмите на кнопку добавления папки . Появится диалог для выбора папки с изображениями. Выберите папку и нажмите кнопку "Ок". Такую операцию можно проделать сколько угодно раз. Для удаления папки из галереи нажмите на кнопку , в диалоге ничего не выбирайте и нажмите "Ок".

Для сравнения между папками нужно добавить папку с изображениями и снизу, и сверху. В случае сравнения в одной галерее, можно добавить папку с помощью любой кнопки.

Шаг 2. Запуск программы

После добавления папок и настройки поиска нажмите на кнопку "Запустить поиск" . Дождитесь завершения поиска в зависимости от количества изображений в папке.

Шаг 3. Окно результатов

В окне результатов будет выведен список изображений, которые имеют похожие изображения и количество подобных. При нажатии кнопки "Просмотр похожих изображений" откроется список подобных изображений с процентом их похожести.
