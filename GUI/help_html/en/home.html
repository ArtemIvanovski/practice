Main Page

The Image Duplicate Finder program is designed to search for similar images as well as their exact duplicates both within a single folder and across multiple folders. The program's algorithm is designed so that different resolutions, color schemes, and compression levels do not affect the comparison quality. The program can process images in various formats. Image Duplicate Finder will find similar images even if one is in BMP format and the other is in standard PNG format. Supported image formats: JPEG, BMP, GIF, PNG.
