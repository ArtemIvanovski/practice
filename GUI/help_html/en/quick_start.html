Quick Start

The program allows for comparing images within one group (gallery) and between two groups of images. A gallery is a group of images that can contain any number of images, even if they are located on different disks.

Comparison Options

Let's consider two comparison options:

  • Within one group

    The program will review all images within the gallery and look for similar ones within the same group. Each image in the gallery will be compared to all other images in that group.

  • Between two groups of images

    The search for similar images will occur not within one group but between galleries. The program will search for similar images from the first group by reviewing all images in the second group. In this case, there is no internal group comparison; each image from the first gallery is compared to each image from the second gallery.

Step 1. Adding Items to Galleries

At this step, you need to select a folder with images. To do this, click the add folder button . A dialog will appear to choose a folder with images. Select the folder and click "OK". You can repeat this operation as many times as needed. To remove a folder from the gallery, click the remove folder button , do not select anything in the dialog, and click "OK".

To compare between folders, you need to add a folder with images both below and above. In the case of comparison within one gallery, you can add a folder using any button.

Step 2. Running the Program

After adding folders and configuring the search, click the "Start Search" button . Wait for the search to complete depending on the number of images in the folder.

Step 3. Results Window

The results window will display a list of images that have similar images and the number of similarities. Clicking the "View Similar Images" button will open a list of similar images with their similarity percentage.
